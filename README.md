# Automed

ML models and datasets for counterfeit drug detection using advanced machine learning techniques.

## Overview

Automed leverages discriminative machine learning models to identify authentic and counterfeit pharmaceuticals with high accuracy.

## Dataset

- **Size**: 6.8 GB
- **Source**: Computational Photography Project For Pill Identification (Data.gov)
- **Training Platforms**: Hugging Face, Teachable Machine
- **Coverage**: 133,000 pill image samples
- **Download**: [Computational Photography Project for Pill Identification](https://catalog.data.gov/dataset/computational-photography-project-for-pill-identification-c3pi-82201/resource/b0250320-b858-47ce-aa4b-976fa02491e0)

## Features

- Discriminative classification approach
- Trained on diverse pharmaceutical samples
- Production-ready model architecture

## Getting Started

For testing this model, a web-based frontend with scanner functionality is available to interact with the cloud-hosted model. See the frontend folder for detailed instructions.
